# 星野の博客 - 个人博客网站

一个温柔可爱风格的个人博客网站，从零手写实现的完整网页。

## ✨ 特性

- 🎨 温柔粉色系配色设计
- 💫 打字机效果标语
- ❄️ Canvas 雪花动画特效
- 📱 响应式移动端适配
- 📬 博客文章模态框展示
- 🌊 平滑滚动导航
- 🎯 丰富的交互动效

## 🛠️ 技术栈

- HTML5 语义化标签
- CSS3 变量、动画、Flexbox 布局
- 原生 JavaScript (ES6+)
- Canvas 动画

## 🚀 使用

直接在浏览器中打开 `index.html` 即可预览。

```bash
# 或使用本地服务器
python -m http.server 8080
# 或
npx serve .
```

## 📂 文件结构

```
├── index.html      # 主页面
├── style.css       # 样式文件
├── main.js         # 交互逻辑
└── README.md       # 项目说明
```

## 🎯 功能板块

1. **个人简介卡片** - 头像、名字、技术背景介绍
2. **实时时间显示** - 动态更新的时间栏
3. **技术栈展示** - 标签云形式展示技能
4. **博客文章列表** - 可点击查看详情
5. **雪花特效** - 增加页面氛围感
